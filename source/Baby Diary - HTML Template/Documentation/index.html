Requirements

Baby Diary HTML template does not require web server to run, except for some sections which require Ajax call, that is Instagram and RSVP form. In order to run these 2 sections, you will need to upload Baby Diary theme to server.You need PHP 5.0+ server and PHP function mail() should be enabled. Please note, without having PHP enabled or just running theme from folder, Baby Diary will work too, except the social feeds and RSVP form.

This template is compatible with following browsers:

  • IE9 +
  • Latest Firefox
  • Latest Opera
  • Latest Google Chrome
  • Latest Safari

Setup

In next sections we will explain how to setup Baby Diary Theme. In order to edit html files you will need a code editor (you can use TextMate or CODA for Mac. or If you have a PC with Windows OS you can use Notepad, Dreamviewer or Sublime Text). Like we explained before, you can view this template in any web browser from your computer, but in order for contact form and portfolio ajax to work, you need to run it from server with PHP installed.

HTML Structure

This theme is built on the custom framework, that is small, fast and with grid system that can be easly changed from 1140px to any other. All grid units (columns) are contained within a top level element, with the class name of container. This has a built in "clearfix" so it does not require a clearing element at the end. It also handles centering the grid layout within the page. By default, it has a max width of 1140px. Between container and columns, you can use row element, that has built in clearfix in case you need to clear elements. Here is the general structure.

<div class="container">
   <div class="row">
      <div class="column six">

      </div>
      <div class="column six">

      </div>
   </div>
</div>

There are twelve different sized columns, each of them with its percentage width:

  .column.one    { width: 8.33333%;}
  .column.two    { width: 16.66667%;}
  .column.three  { width: 25%;}
  .column.four   { width: 33.33333%;}
  .column.five   { width: 41.66667%;}
  .column.six    { width: 50%;}
  .column.seven  { width: 58.33333%;}
  .column.eight  { width: 66.66667%;}
  .column.nine   { width: 75%;}
  .column.ten    { width: 83.33333%;}
  .column.eleven { width: 91.66667%;}
  .column.twelve { width: 100%;}

so for example, if you want to split container in half you would use two "column six" columns:

<div class="container">
   <div class="row">
      <div class="column six">

      </div>
      <div class="column six">

      </div>
   </div>
</div>
Each column unit has 15px of padding on the right and left side that, when coupled with the padding from an adjacent column unit (or the grid-container itself), creates a 30px gutter.

Instagram



	In order for Instagram to work, you need to edit custom.js file, line 305 - Instagram Section.



	You have just two lines that you need to change:


        var instagramTag = 'baby'; // Add Instagram Tag here
        var instagramClientId = '630b5c9d2cab44e08fb9d014cf00b2b6'; // Add ClientId here



	First one is instagram tag that you want images to appear. Second one is Instagram Client ID. 



	In order to find Instagram Client ID, follow these steps:



	1. Login if your not already.



	2. Go to the Instagram Developer page.



	3. Click on ”Register your application”.



	4. If you don’t already have a application registered, you will now see a form to register a new one. If you instead see a list of ”clients”, click the ”Register new client” at the top right.



	5. Set application name to your websites name



	6. Type in a description, anything you want



	7. Add the url of you site on the two last fields.



	8. Hit register.



	9. Click ”Manage clients” in the top menu and you will now see you new client in the list and a client id, client secret and more. Copy the client_id and past that in custom.js file.



	10. Save custom.js and thats it.

	Hero Background



	To change image in this section, open up style.css and search for "hero.background" class. Just change background property with yours and that should be it.

	To change image in this section, open up style.css and search for "parallax" class. Just change background-image property with yours and that should be it. Pretty easy, right :)
